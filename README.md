## Git

### 🌿 Branch

- main : 서비스 운영 브랜치
- dev : 개발중인 브랜치
- feat/기능명 : 기능 개발 브랜치

<br/>

### 💌 Commit Message

| Message  |                  설명                  |
| :------: | :------------------------------------: |
|   feat   |            새로운 기능 추가            |
|   fix    |               버그 수정                |
|   docs   |               문서 수정                |
| refactor |             코드 리팩터링              |
|  style   | 코드 의미에 영향을 주지 않는 변경사항  |
|   test   | 테스트 코드, 리팩터링 테스트 코드 추가 |
|  chore   | 빌드 부분 혹은 패키지 매니저 수정사항  |
|  rename  |      파일 혹은 폴더명을 수정사항       |
| release  |                  배포                  |
